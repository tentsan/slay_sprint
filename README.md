# SLAY SPRINT

Slay the Spire にインスパイアされた、**3〜5分で遊べるオートバトルRPG**です。
プレイヤーは自動で戦う勇者を見守りながら、バトルの合間に報酬を選んで強化していきます。
全10戦を勝ち抜いて、最後のボスを倒しましょう。

## 遊び方

### ブラウザで遊ぶ

GitHub Pages にデプロイ済みの場合は、公開URLにアクセスするだけで遊べます。

### ローカルで遊ぶ

```bash
git clone https://github.com/tentsan/slay_sprint.git
cd slay_sprint
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。

> **注意:** `index.html` を直接ダブルクリックで開くと、CSVの読み込みに失敗します。必ずローカルサーバー経由でアクセスしてください。

## ゲームの流れ

```
タイトル画面 → バトル → 報酬選択 → バトル → 報酬選択 → ... → 10戦目（ボス） → 結果画面
```

1. **START GAME** を押すとゲーム開始
2. バトルは全自動で進行（見ているだけでOK）
3. 勝利すると **3つの報酬** から1つを選択
4. 選んだ報酬でキャラクターが強化され、次のバトルへ
5. **10戦クリア** でゲームクリア、途中で倒されるとゲームオーバー
6. 結果画面から **BACK TO TITLE** でタイトルに戻り、何度でも遊べます

## バトルシステム

バトルは**カードベースのターン制オートバトル**です。

### ターンの流れ

毎ターン以下の順番で進行します。

1. **プレイヤーのカードプレイ** — 手持ちのカードから1枚を自動選択して使用
2. **敵のカードプレイ** — 敵も同様にカードを1枚使用

プレイヤーが常に先攻です。

### カードシステム

各キャラクターは**カード**を持っており、バトル中は毎ターン自動的に1枚選ばれてプレイされます。
カードにはエフェクト（`damage`、`heal` 等）が定義されており、プレイ時に順番に解決されます。

現在は全キャラクター共通で以下の1枚のみです。

| カード | 効果 |
|--------|------|
| **Deal 5 Damage** | 敵に 5 ダメージを与える |

> 今後、カードの種類を増やし、エネミーごとに異なるカードを設定できるよう拡張予定です。

## 報酬システム

バトルに勝利するたびに、3つの報酬カードから1つを選べます。

| 報酬 | 種別 | 効果 |
|------|------|------|
| **Health Potion** 🧪 | 即時 | HP を 20 回復 |
| **Full Restore** ✨ | 即時 | HP を全回復 |
| **Heart Crystal** ❤️ | 永続 | 最大HP +10（現在HPも同量回復） |

バトル間のHP管理が戦略のカギになります。回復を取るか、最大HPを伸ばして将来に備えるかの選択です。

## 敵の難易度

敵はHPに基づいて3つのティアに分けられ、バトルが進むほど強い敵が登場します。

| バトル | 難易度 |
|--------|--------|
| 1〜3戦目 | HP が低い敵（スライム、ゴブリンなど） |
| 4〜6戦目 | HP が中程度の敵（ナイト、ゴーレムなど） |
| 7〜9戦目 | HP が高い敵（ドラゴンなど） |
| 10戦目 | **ボス**（最もHPが高い敵が確定で登場） |

## データのカスタマイズ

ゲームのデータはCSVファイルと JavaScript で管理されています。

```
data/
├── enemies.csv   # 敵の名前・絵文字・HP
├── rewards.csv   # 報酬の種類・効果
└── player.csv    # プレイヤーの名前・絵文字・HP

js/
└── cards.js      # カード定義（エフェクト・ダメージ値など）
```

敵のHPを変えたい場合は `data/enemies.csv` を編集してください。
カードの効果やダメージ値を変えたい場合は `js/cards.js` を編集してください。
