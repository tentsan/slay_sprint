# SLAY SPRINT

Slay the Spire にインスパイアされた、**3〜5分で遊べるオートバトルRPG**です。
プレイヤーは自動で戦う勇者を見守りながら、バトルの合間に報酬を選んで強化していきます。
全10戦を勝ち抜いて、最後のボスを倒しましょう。

## 遊び方

### ブラウザで遊ぶ

GitHub Pages にデプロイ済みの場合は、公開URLにアクセスするだけで遊べます。

### ローカルで遊ぶ

```bash
git clone https://github.com/tentsan/slay_sprint.git
cd slay_sprint
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。

> **注意:** `index.html` を直接ダブルクリックで開くと、CSVの読み込みに失敗します。必ずローカルサーバー経由でアクセスしてください。

## ゲームの流れ

```
タイトル画面 → バトル → 報酬選択 → バトル → 報酬選択 → ... → 10戦目（ボス） → 結果画面
```

1. **START GAME** を押すとゲーム開始
2. バトルは全自動で進行（見ているだけでOK）
3. 勝利すると **3つの報酬** から1つを選択
4. 選んだ報酬でキャラクターが強化され、次のバトルへ
5. **10戦クリア** でゲームクリア、途中で倒されるとゲームオーバー
6. 結果画面から **BACK TO TITLE** でタイトルに戻り、何度でも遊べます

## バトルシステム

バトルはターン制のオートバトルです。毎ターン以下の流れで進行します。

### ターンの流れ

1. **リジェネ** — プレイヤーにリジェネがあればHP回復
2. **状態異常ダメージ** — 毒・火傷のダメージが発生
3. **攻撃** — 速度（SPD）が高い方が先に攻撃
4. **特殊能力** — 敵の特殊能力が確率で発動

### 攻撃タイプ

バトルには**物理攻撃**と**魔法攻撃**の2種類があります。

- **物理攻撃**: ATK（攻撃力）で攻撃し、相手の DEF（防御力）で軽減
- **魔法攻撃**: INT（魔法攻撃力）で攻撃し、相手の MEN（魔法防御力）で軽減

**敵の攻撃タイプ**はCSVで敵ごとに設定されています。魔女・ドラゴン・デーモンは魔法型です。

**プレイヤーの攻撃タイプ**は ATK と INT を比較して、高い方を自動選択します。
初期状態では ATK(8) > INT(5) のため物理攻撃ですが、報酬で INT を上げると魔法型に転向できます。

### ダメージ計算

```
物理ダメージ = ATK − 相手のDEF　※最低1ダメージ保証
魔法ダメージ = INT − 相手のMEN　※最低1ダメージ保証
```

- **クリティカル**: プレイヤーは一定確率で1.5倍ダメージ（物理・魔法共通）
- **棘（Thorns）**: 物理攻撃を受けた側が反射ダメージを与える（魔法攻撃には反応しない）
- **吸血（Lifesteal）**: 与えたダメージの一部でHP回復（物理・魔法共通）

## 敵の特殊能力

敵はそれぞれ固有の特殊能力を持っています（発動は確率）。

| 能力 | 効果 |
|------|------|
| **毒（Poison）** | 毎ターン3ダメージ × 3ターン |
| **火傷（Burn）** | 毎ターン4ダメージ × 2ターン |
| **吸血（Lifesteal）** | 攻撃力の50%分のHPを回復 |
| **弱体化（Weaken）** | 相手が使用中の攻撃ステータスを2下げる（物理型ならATK、魔法型ならINT） |
| **シールド（Shield）** | 自身の防御力を5上げる |
| **ドレイン（Drain）** | 相手から5HP吸収 |

> 弱体化・シールドの効果はそのバトル限りで、次の戦闘には持ち越されません。

## 報酬システム

バトルに勝利するたびに、ランダムな3つの報酬カードから1つを選べます。

### 即時効果

その場で効果を発揮し、消費されます。

- **Health Potion** 🧪 — HP を 20 回復
- **Full Restore** ✨ — HP を全回復

### 永続強化

選ぶとゲーム終了まで効果が持続します。重複取得で効果が積み重なります。

- **攻撃力UP** 🗡️⚔️ — ATK +2 / +4
- **防御力UP** 🛡️🏰 — DEF +2 / +4
- **魔法攻撃力UP** 📘🔮 — INT +2 / +4
- **魔法防御力UP** 🧿👘 — MEN +2 / +4
- **速度UP** 👢 — SPD +1
- **最大HP UP** ❤️ — 最大HP +10（現在HPも同量回復）
- **リジェネ** 💎 — 毎ターンHP 3 回復
- **クリティカル率UP** 🍀 — クリティカル率 +15%
- **棘** 🦔 — 被弾時に敵へ 3 ダメージ（物理攻撃のみ）
- **吸血** 🧛 — 与ダメージの 10% を回復

## 敵の難易度

敵は難易度に応じて3つのティアに分けられ、バトルが進むほど強い敵が登場します。

| バトル | 難易度 |
|--------|--------|
| 1〜3戦目 | 弱い敵（スライム、ゴブリンなど） |
| 4〜6戦目 | 中程度の敵（ゴーレム、魔女など） |
| 7〜9戦目 | 強い敵（ドラゴンなど） |
| 10戦目 | **ボス**（最強の敵が確定で登場） |

## データのカスタマイズ

ゲームのデータはすべて CSV ファイルで管理されています。
Google スプレッドシート等で編集して、バランス調整が可能です。

```
data/
├── enemies.csv   # 敵のステータス・特殊能力
├── rewards.csv   # 報酬の種類・効果
└── player.csv    # プレイヤーの初期ステータス
```

例えば、プレイヤーを強くしたい場合は `data/player.csv` の `attack` の値を上げるだけでOKです。
